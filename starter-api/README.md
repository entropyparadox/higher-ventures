# Starter API

EP 프로젝트 개발에 사용되는 API 보일러플레이트 입니다.
NestJS 프레임워크의 [monorepo](https://docs.nestjs.com/cli/monorepo#monorepo-mode) 구조로 구성되어 있으며,
여러 프로젝트에서 많이 사용되는 공통 기능들을 일부 포함하고 있습니다.

## 사용 버전

- NestJS 8.x
- TypeORM 0.2
- NodeJS 16.14.2, NPM 8.5.5
- MySQL 8
